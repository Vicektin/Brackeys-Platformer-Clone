# My First Platformer Game (Godot 4)

This is my very first game project! I created this 2D platformer by following the tutorial series by Brackeys.

## 🎮 How to Play

* **Move:** Arrow Keys / WASD
* **Jump:** Space Bar
* **Objective:** Collect coins and don't die!

## 🛠️ Built With

* **Engine:** Godot 4.5
* **Language:** GDScript
